## 游戏简介

### 1.1 游戏概述

《石头人保卫战》是一款融合塔防和卡牌元素的微信小游戏。你将扮演一位指挥官，通过收集和使用各类怪物卡片，在战场上与敌方AI进行实时对战，保护己方城堡免受敌方攻击。

### 1.2 游戏特色

- **丰富的卡牌系统**：15种不同属性的怪物卡片，每张卡片具有独特的属性和技能
- **实时对战体验**：与AI敌手进行实时战斗，考验你的策略和反应能力
- **多关卡难度**：从简单到困难的多个关卡，逐步提升游戏难度
- **微信集成**：完整的微信用户认证，支持排行榜和成就系统
- **精美画面**：高质量的游戏画面和流畅的动画效果
- **丰富音效**：沉浸式的背景音乐和音效设计

### 1.3 系统要求

| 项目 | 要求 |
|------|------|
| **微信版本** | 7.0 及以上 |
| **操作系统** | iOS 12.0+ 或 Android 5.0+ |
| **屏幕分辨率** | 1080x1920 或更高 |
| **网络连接** | 需要网络连接 |
| **存储空间** | 至少 50MB 可用空间 |

## 快速开始

### 2.1 安装游戏

1. **打开微信**
   - 在微信中搜索"石头人保卫战"
   - 或扫描游戏二维码

2. **进入游戏**
   - 点击"进入小程序"
   - 等待游戏加载完成

3. **授权用户信息**
   - 首次进入时，游戏会请求获取你的微信用户信息
   - 点击"允许"以获得最佳游戏体验
   - 你的信息仅用于游戏内显示，不会被泄露

### 2.2 首次游戏

1. **查看主菜单**
   - 游戏加载完成后，你会看到主菜单
   - 菜单上显示你的微信昵称和头像

2. **选择游戏模式**
   - 点击"开始游戏"进入关卡选择
   - 选择你想要挑战的关卡

3. **进入游戏**
   - 点击关卡后，游戏会加载战场
   - 等待加载完成后，游戏开始

### 2.3 基本操作

| 操作 | 说明 |
|------|------|
| **点击卡牌** | 使用卡牌召唤怪物 |
| **点击设置** | 打开设置面板 |
| **点击帮助** | 查看游戏帮助 |
| **点击返回** | 返回上一个界面 |

## 游戏规则

### 3.1 基本规则

#### 3.1.1 游戏目标

- **保护城堡**：保护己方城堡不被敌方怪物摧毁
- **摧毁敌堡**：摧毁敌方城堡以获得胜利

#### 3.1.2 胜负条件

| 条件 | 结果 |
|------|------|
| **己方城堡血量 ≤ 0** | 游戏失败 |
| **敌方城堡血量 ≤ 0** | 游戏胜利 |
| **时间限制** | 超时则判定为失败 |

### 3.2 法力系统

#### 3.2.1 法力机制

- **初始法力**：游戏开始时，你拥有一定的初始法力
- **法力恢复**：每秒恢复一定量的法力
- **法力消耗**：使用卡牌时消耗相应的法力
- **法力上限**：法力有最大值限制

#### 3.2.2 法力管理

```
法力值显示
    │
    ├─ 当前法力 / 最大法力
    │
    ├─ 法力条（进度条）
    │
    └─ 法力恢复速度：每秒 +1
```

### 3.3 卡牌系统

#### 3.3.1 卡牌属性

每张卡牌都有以下属性：

| 属性 | 说明 |
|------|------|
| **卡牌名称** | 怪物的名称 |
| **法力消耗** | 使用卡牌需要消耗的法力 |
| **怪物等级** | 怪物的初始等级 |
| **卡牌图标** | 怪物的外观图标 |

#### 3.3.2 卡牌状态

| 状态 | 说明 | 外观 |
|------|------|------|
| **可用** | 法力充足，可以使用 | 正常显示 |
| **禁用** | 法力不足，无法使用 | 灰色显示 |
| **冷却** | 刚使用过，等待冷却 | 显示冷却进度 |

### 3.4 怪物系统

#### 3.4.1 怪物属性

每个怪物都有以下属性：

| 属性 | 说明 |
|------|------|
| **血量** | 怪物的生命值 |
| **攻击力** | 怪物对敌方造成的伤害 |
| **攻击速度** | 怪物的攻击频率 |
| **移动速度** | 怪物的移动速度 |
| **攻击范围** | 怪物的攻击距离 |

#### 3.4.2 怪物行为

1. **生成**：使用卡牌后，怪物在指定位置生成
2. **移动**：怪物自动向敌方城堡移动
3. **攻击**：进入攻击范围后，怪物自动攻击敌方
4. **死亡**：血量降至0时，怪物死亡并消失

### 3.5 AI系统

#### 3.5.1 敌方AI

敌方AI会根据当前局势自动做出决策：

- **防守策略**：当你的怪物威胁度高时，敌方会优先出防御型怪物
- **进攻策略**：当敌方优势时，敌方会优先出高伤害怪物
- **平衡策略**：根据局势动态调整出牌策略
- **法力管理**：敌方会优先出高性价比卡牌

#### 3.5.2 AI难度

| 难度 | 说明 | 敌方AI表现 |
|------|------|-----------|
| **简单** | 初级玩家 | AI决策较慢，策略简单 |
| **普通** | 中级玩家 | AI决策适中，策略均衡 |
| **困难** | 高级玩家 | AI决策迅速，策略复杂 |

## 游戏策略

### 6.1 基础策略

#### 6.1.1 法力管理

- **合理分配**：不要一次性消耗所有法力
- **保留储备**：始终保留一些法力用于应急
- **优先级**：根据局势选择优先出牌的卡牌

#### 6.1.2 卡牌选择

- **了解卡牌**：熟悉每张卡牌的属性和特点
- **灵活搭配**：根据敌方阵容调整自己的卡牌
- **多样化**：不要过度依赖某一张卡牌

#### 6.1.3 时机把握

- **及时防守**：当敌方怪物接近时，及时出防御型怪物
- **趁势进攻**：当敌方防御薄弱时，集中出攻击型怪物
- **平衡发展**：在防守和进攻之间找到平衡点

### 6.2 高级策略

#### 6.2.1 局势评估

```
评估要点：
1. 敌方城堡血量
   - 血量低 → 加强进攻
   - 血量高 → 稳定防守

2. 己方城堡血量
   - 血量低 → 加强防守
   - 血量高 → 可以进攻

3. 敌方怪物数量
   - 数量多 → 出防御型
   - 数量少 → 出进攻型

4. 当前法力
   - 法力多 → 可以出高消耗卡牌
   - 法力少 → 出低消耗卡牌
```

#### 6.2.2 阵容克制

```
克制关系：
- 防御型 克制 进攻型（吸收伤害）
- 进攻型 克制 远程型（快速接近）
- 远程型 克制 防御型（远程输出）
- 特殊型 克制 所有类型（特殊效果）
```

#### 6.2.3 节奏控制

```
游戏节奏：
1. 开局阶段（0-30秒）
   - 稳定防守
   - 积累法力
   - 观察敌方

2. 中期阶段（30-120秒）
   - 平衡防守和进攻
   - 根据局势调整
   - 积累优势

3. 后期阶段（120秒+）
   - 集中进攻或防守
   - 根据血量差距调整
   - 争取胜利
```

### 6.3 常见错误

| 错误 | 后果 | 解决方案 |
|------|------|---------|
| **过度消耗法力** | 无法应急 | 保留法力储备 |
| **忽视防守** | 城堡被摧毁 | 及时出防御型 |
| **过度防守** | 无法进攻 | 平衡防守和进攻 |
| **卡牌搭配不当** | 效率低下 | 学习克制关系 |
| **节奏把握不好** | 局势被动 | 观察敌方节奏 |

A: 
1. 检查网络连接

3. 清除微信缓存
4. 重新启动游戏

A:

2. 关闭背景音乐
